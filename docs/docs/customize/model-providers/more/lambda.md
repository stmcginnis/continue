# Lambda

Lambda provides high performance and affordable inferencing services.

## Config Example

```json title="config.json"
{
  "models": [
    {
      "title": "Deepseek Coder",
      "provider": "lambda",
      "model": "deepseek-coder-v2-lite-instruct",
      "apiKey": "<LAMBDA_API_KEY>"
    }
  ]
}
```
